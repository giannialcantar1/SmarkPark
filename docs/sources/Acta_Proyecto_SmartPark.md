# Acta de Proyecto
## SmartPark - Sistema de Gestion de Estacionamientos

### 1. Datos generales
- Fecha: 25 de abril de 2026
- Version del documento: 1.0
- Proyecto: SmartPark - Sistema de Gestion de Estacionamientos

### 2. Objetivo general
Disenar e implementar un sistema web para administrar de forma eficiente los procesos operativos de un estacionamiento, incluyendo autenticacion, control de acceso, gestion de espacios, registro de vehiculos, cobros y reportes.

### 3. Objetivos especificos
- Centralizar la administracion del garage en una plataforma web
- Automatizar el registro de entradas y salidas
- Controlar la ocupacion de espacios en tiempo real
- Registrar y consultar vehiculos por usuario y garage
- Procesar pagos y consultar recibos
- Generar reportes operativos de ocupacion e ingresos
- Asegurar trazabilidad mediante logs y alertas de acceso

### 4. Alcance
El proyecto cubre:
- backend Flask con arquitectura POO
- integracion con Supabase
- autenticacion con JWT
- frontend React + Vite
- panel administrativo y operativo
- documentacion tecnica y de usuario

No incluye:
- integracion con pasarelas bancarias reales
- despliegue productivo administrado
- hardware fisico de acceso automatizado

### 5. Beneficiarios

- operadores de acceso
- usuarios propietarios de vehiculos
- responsables financieros y de control
