# Acta de Proyecto - SmartPark

## 1. Datos generales

- Nombre del proyecto: SmartPark
- Tipo de proyecto: Sistema de Gestion de Estacionamientos
- Institucion: Instituto Politecnico Parroquial Santa Ana
- Fecha: 2026

## 2. Descripcion del proyecto

SmartPark es una plataforma web diseñada para optimizar la administracion de estacionamientos mediante el control digital de usuarios, vehiculos, espacios, entradas, salidas, pagos y reportes. El sistema integra autenticacion segura con JWT y verificacion OTP por correo, una API backend en Flask, una interfaz frontend en React y almacenamiento en Supabase PostgreSQL.

## 3. Objetivo general

Desarrollar un sistema de gestion de estacionamientos que permita automatizar el control operativo del garaje, mejorar la trazabilidad de los vehiculos y facilitar la toma de decisiones a traves de metricas, pagos y reportes.

## 4. Objetivos especificos

- Implementar registro e inicio de sesion seguro para usuarios del sistema.
- Permitir la gestion de vehiculos asociados a cada usuario.
- Administrar la disponibilidad de espacios de parqueo por piso.
- Registrar entradas y salidas de vehiculos en tiempo real.
- Calcular y registrar pagos de forma automatizada.
- Generar reportes de ocupacion, ingresos, vehiculos y usuarios.
- Integrar un modulo de alertas y notificaciones para eventos relevantes.

## 5. Alcance

El proyecto cubre:

- autenticacion con JWT y OTP
- gestion de usuarios y roles
- gestion de vehiculos
- control de espacios de parqueo
- registro de entradas y salidas
- procesamiento de pagos
- dashboard administrativo
- reportes y consultas operativas
- notificaciones y alertas de acceso

No incluye en esta fase integraciones con hardware fisico de barreras, sensores o lectores externos en produccion.

## 6. Tecnologias utilizadas

- Frontend: React + Vite
- Backend: Python + Flask
- Base de datos: Supabase PostgreSQL
- Autenticacion: JWT
- Verificacion adicional: OTP por Gmail SMTP

## 7. Beneficios esperados

- Mayor control sobre la ocupacion del estacionamiento
- Reduccion de errores manuales
- Mejor seguimiento de pagos y movimientos
- Mayor seguridad en el acceso al sistema
- Disponibilidad de informacion centralizada para administracion
