# Cronograma del Proyecto SmartPark

## Resumen general

- Proyecto: SmartPark
- Periodo de ejecucion: enero 2026 a mayo 2026
- Modalidad: desarrollo incremental por fases

## Fases del proyecto

1. Analisis de requerimientos
2. Diseno de base de datos
3. Desarrollo del backend (Flask)
4. Desarrollo del frontend (React)
5. Integracion y pruebas
6. Documentacion
7. Entrega final
